# Java Data Structures and Algorithm Practice

This repository contains a collection of Java programming problems and solutions focused on improving problem-solving skills and strengthening concepts in Data Structures and Algorithms.

The programs are implemented using Java and cover important topics frequently asked in coding interviews and technical assessments.

## Topics Covered

### Arrays
This section contains around **50 array-based problems**, including:

- Find minimum and maximum element in array
- Remove duplicates from sorted array
- Union and intersection of two arrays
- Find missing number in array
- Move negative numbers to one side
- Sliding window problems
- Subarray problems
- Two pointer techniques

### Strings
This section contains **50 string problems**, including:

- Reverse a string
- Reverse each word in a string
- Longest common prefix
- Check palindrome
- Anagram checking
- Remove special characters
- Character frequency count
- Longest substring problems

### Stack Patterns
Implementation of **4 stack-based patterns/problems**, including:

- Stack implementation using array
- Balanced parentheses problem
- Next greater element
- Stack-based pattern questions

### Heap Patterns
Contains **2 heap-based problems**, including:

- Basic heap implementation
- Priority queue / heap based problem solving

### Additional Problems
Some additional **array and string problems** are also included for extra practice and concept clarity.

## Technologies Used

- Java
- Object Oriented Programming (OOP)
- Data Structures
- Algorithms
- Basic Problem Solving Techniques

## Purpose of This Repository

The goal of this repository is to:
